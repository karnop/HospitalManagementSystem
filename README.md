# HospitalManagementSystem
Simple Hospital Management System using Java, JDBC and MySql

## Functionality:  

#### HospitalManagement.java :  
* The program start here
* contains an instance of doctor and patient class 
* void bookAppointment() : appointments are booked between doctor and patient  

#### Doctor.java :
* void viewDoctors() : print a list of all doctors (the doctor table)
* boolean getDoctorById(id) : check if a doctor with a given id exists

#### Patient.java : 
* void addPatient() : add a patient to the patient table in the hospital database
* void viewPatients() : print a list of all patients (the patients table)
* boolean getPatientById(id) : check if a patient with a given id exists
